# Dashing Arrivals!

Use Case: Imagine you're in the city of Seattle and you'd like to immediately find the nearest transit stops and what arrivals are coming into the station! It seems like a simple task, but there aren't very many apps out there that actually provide this use case! Wild!

## The First Phase - Step 1

*Automatically finding your location via GPS or enter an address.*

Drilling into the specifics, the idea is the application would allow via a web application, the ability to open the site/application via https://dashingarrivals.com and up pops the site which finds your immediate location (obviously you'd have to allow it). That location could be countered by clicking on button at the bottom of the app, that would allow a specific address to be entered instead of the automatically determined GPS location.
